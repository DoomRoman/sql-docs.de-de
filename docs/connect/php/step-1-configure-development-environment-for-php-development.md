# <a name="step-1-configure-environment-for-php-development"></a>Schritt 1: Konfigurieren der Entwicklungsumgebung für PHP-Entwicklung
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* Identifiziert die Version des Treibers PHP Sie verwenden möchten, basierend auf Ihrer Umgebung, wie hier beschrieben: [Systemanforderungen für Microsoft Drivers for PHP for SQL Server](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* Herunterladen und Installieren der anwendbaren PHP-Treiber hier: [Microsoft PHP-Treiber herunterladen](https://www.microsoft.com/download/details.aspx?id=20098)  
* Herunterladen und installieren Sie die entsprechenden ODBC-Treiber hier: [ODBC-Treiber für SQL Server herunterladen](../../connect/odbc/download-odbc-driver-for-sql-server.md)  
* Konfigurieren der PHP-Treiber und die Web-Server für Ihr spezifisches Betriebssystem:

### <a name="windows"></a>Windows  
  

* Konfigurieren den PHP-Treiber zu laden, folgendermaßen: [Loading the Microsoft Drivers for PHP für SQL Server](../../connect/php/loading-the-php-sql-driver.md) 
* Konfigurieren von IIS zum Hosten von PHP Anwendungen, wie hier beschrieben: [Konfigurieren von IIS für die Microsoft Drivers for PHP for SQL Server](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux und Mac


*   Konfigurieren den PHP-Treiber zu laden und Ihren Webserver zum Hosten von PHP Anwendungen, zu konfigurieren, wie hier beschrieben: [PHP Linux- und Mac-Treiber-Installation-Tutorial](../../connect/php/installation-tutorial-linux-mac.md)
