# <a name="sqlserverclob-class"></a>SQLServerClob-Klasse
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Stellt ein CLOB (Character Large Binary Object) dar.  
  
 **Paket:** com.microsoft.sqlserver.jdbc  
  
 **Erweitert:** java.lang.Object  
  
 **Implementiert** java.sql.Clob und java.io.Serializable  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public class SQLServerClob  
```  
  
## <a name="remarks"></a>Remarks  
 In [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] wird ein CLOB als TEXT- oder NTEXT-Datentyp gespeichert.
