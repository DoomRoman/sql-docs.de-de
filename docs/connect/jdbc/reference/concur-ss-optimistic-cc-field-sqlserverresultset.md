# <a name="concurssoptimisticcc-field-sqlserverresultset"></a>CONCUR_SS_OPTIMISTIC_CC-Feld (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Dient zum Angeben eines Typs der optimistischen Nebenläufigkeit (Lese-/Schreibzugriff) für [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] ohne Zeilensperre.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CC  
```  
  
## <a name="field-value"></a>Feldwert  
 Ein **Int** Wert von 1008.
