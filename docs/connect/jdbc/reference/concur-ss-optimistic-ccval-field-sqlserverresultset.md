# <a name="concurssoptimisticccval-field-sqlserverresultset"></a>CONCUR_SS_OPTIMISTIC_CCVAL-Feld (SQLServerResultSet)

```  
  
public static final int CONCUR_SS_OPTIMISTIC_CCVAL  
```
