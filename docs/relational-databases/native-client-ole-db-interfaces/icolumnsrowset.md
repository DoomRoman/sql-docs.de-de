[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client fügt IColumnsRowset::GetColumnRowset die Spalte DBCOLUMN_BASETABLEINSTANCE hinzu.
