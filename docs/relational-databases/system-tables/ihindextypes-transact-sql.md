# <a name="ihindextypes-transact-sql"></a>IHindextypes (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Die **IHindextypes** -Systemtabelle enthält eine Zeile für jeden nicht - SQL Server-Indextyp für nicht - SQL Server-Verleger unterstützt. Diese Tabelle wird in der Verteilungsdatenbank gespeichert.  
  
|Spaltenname|Datentyp|Description|  
|-----------------|---------------|-----------------|  
|**type**|**nvarchar(255)**|Der Name eines unterstützten nicht - SQL Server-Index-Typs.|
