# <a name="translator-setup-dlls"></a>Setup-DLLs für das Konvertierungsprogramm
> [!NOTE]  
>  Ab Windows XP und Windows Server 2003, ist ODBC in das Windows-Betriebssystem enthalten. Sie sollten nur explizit ODBC in früheren Versionen von Windows installieren.  
  
 Die Translator-Einrichtung DLL enthält die **ConfigTranslator** Funktion, die die Standardoption für einen Übersetzer zurückgibt. Falls erforderlich, werden diese Informationen vom Benutzer aufgefordert. Eine vollständige Beschreibung dieser Funktion, finden Sie unter [-Setup-DLL-API-Referenz](../../../odbc/reference/syntax/setup-dll-api-reference.md).  
  
 Die Translator-Setup wird die DLL der Translator-Entwickler geschrieben. Teil des Übersetzers möglich DLL oder einer separaten DLL.
