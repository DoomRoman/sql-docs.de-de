# <a name="general-error-checks"></a>Überprüfungen auf allgemeine Fehler
Der Treiber-Manager überprüft, ein allgemeinen Fehler. Wird immer SQL_ERROR zurückgegeben, wenn die folgenden Fehler festgestellt: die Funktion, die vom Treiber unterstützt werden muss.
