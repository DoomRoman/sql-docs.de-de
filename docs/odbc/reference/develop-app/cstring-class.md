# <a name="cstring-class"></a>CString-Klasse
Da Objekte von der **CString** Klasse in Microsoft® Visual C++® werden signiert und Zeichenfolgenargumente in ODBC-Funktionen sind ohne Vorzeichen, Anwendungen, die verstreichen **CString** Objekte mit ODBC-Funktionen ohne Wandeln sie erhalten Compiler-Warnungen.
