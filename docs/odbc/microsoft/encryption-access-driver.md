(Dieses Steuerelement ist nur verfügbar, wenn Version 4. *x* Format ausgewählt ist.)  
  
 Mit der Veröffentlichung der Microsoft Jet-Datenbank-Engine-Version 4.0 ist die datenbankverschlüsselung zulässig, wenn es sich bei Access-Datenbanken zu erstellen. Finden Sie unter den *Jet-Datenbank-Engine-Programmierreferenz* für zusätzliche Informationen.
