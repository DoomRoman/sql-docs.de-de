# <a name="index-name-limitations"></a>Einschränkungen für Indexnamen
Wenn die Paradox-Treiber verwendet wird, benötigen ein primärer Index den gleichen Namen wie die Tabelle auf dem sie definiert ist. Nicht eindeutige oder nicht eindeutige Indizes müssen den gleichen Namen wie die Tabelle auf denen sie definiert sind.
