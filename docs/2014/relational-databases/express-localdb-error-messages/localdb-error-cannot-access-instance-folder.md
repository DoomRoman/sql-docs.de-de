|-|-|  
|Produktname|SQL Server|  
|Ereignis-ID|261|  
|Ereignisquelle|Lokale SQL Server-Datenbanklaufzeit 12.0|  
|Komponente|Laufzeit-API der lokalen Datenbank|  
|Meldungstext|Lokale Datenbank instanzordner kann nicht zugegriffen werden: %%LOCALAPPDATA%%\Microsoft\Microsoft SQL Server lokale DB\Instances\\< Instanzname\>.|
