# <a name="localdberrorinvalidinstancename"></a>LOCALDB_ERROR_INVALID_INSTANCE_NAME
    
## <a name="details"></a>Details  
  
|||  
|-|-|  
|Produktname|SQL Server|  
|Ereignis-ID|283|  
|Ereignisquelle|Lokale SQL Server-Datenbanklaufzeit 12.0|  
|Komponente|Laufzeit-API der lokalen Datenbank|  
|Meldungstext|Der Instanzname enthält ungültige Zeichen oder ist zu lang. Informieren Sie sich in der API-Dokumentation.|  
  
## <a name="explanation"></a>Erklärung  
 Der Instanzname enthält ungültige Zeichen oder ist zu lang.  
  
## <a name="user-action"></a>Benutzeraktion  
 Informieren Sie sich in der lokalen Datenbanklaufzeit-API-Dokumentation über die gültigen Namen für lokale Datenbanklaufzeitinstanzen.
