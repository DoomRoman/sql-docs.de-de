# <a name="serversettingsgeneralflag-class"></a>ServerSettingsGeneralFlag-Klasse
  Die [ServerSettingsGeneralFlag-Klasse](serversettingsgeneralflag-class.md) stellt die allgemeinen Flageinstellungen in einer Instanz von [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]dar.  
  
 Mit dieser Klasse können Sie folgende Vorgänge ausführen:  
  
-   Anzeigen des Flagnamens.  
  
-   Anzeigen und Ändern des Flagwerts.
