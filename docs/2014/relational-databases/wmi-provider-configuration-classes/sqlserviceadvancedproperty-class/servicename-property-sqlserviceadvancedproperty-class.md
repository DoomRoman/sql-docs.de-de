## <a name="syntax"></a>Syntax  
  
```  
  
object  
.ServiceName [= value]  
```

 Ein Objekt der [SqlServiceAdvancedProperty-Klasse](sqlserviceadvancedproperty-class.md) , das eine erweiterte Eigenschaft darstellt.
