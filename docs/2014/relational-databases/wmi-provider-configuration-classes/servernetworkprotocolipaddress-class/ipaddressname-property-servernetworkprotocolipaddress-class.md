# <a name="ipaddressname-property-servernetworkprotocolipaddress-class"></a>IpAddressName-Eigenschaft (ServerNetworkProtocolIPAddress-Klasse)
  Ruft den Namen der IP-Adresse ab.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
object  
.IpAddressName [= value]  
```  
  
## <a name="parts"></a>Teile  
 *object*  
 Ein Objekt der [ServerNetworkProtocolIPAdress-Klasse](servernetworkprotocolipaddress-class.md) , das eine IP-Adresse für das Netzwerkprotokoll in der Instanz von [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]darstellt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Ein Zeichenfolgenwert, der den Namen der IP-Adresse angibt.
