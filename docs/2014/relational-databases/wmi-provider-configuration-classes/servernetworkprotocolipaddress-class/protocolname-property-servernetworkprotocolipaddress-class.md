# <a name="protocolname-property-servernetworkprotocolipaddress-class"></a>ProtocolName-Eigenschaft (ServerNetworkProtocolIPAddress-Klasse)
  Ruft den Namen des Servernetzwerkprotokolls ab, das auf die IP-Adresse verweist.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Teile  
 *object*  
 A [ServerNetworkProtocolIPAdress-Klassenobjekt](servernetworkprotocolipaddress-class.md) , das eine IP-Adresse für das Netzwerkprotokoll in der Instanz von [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]darstellt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Ein Zeichenfolgenwert, der den Namen des Server-Netzwerkprotokolls angibt.
