# <a name="instancename-property-servernetworkprotocolproperty-class"></a>InstanceName-Eigenschaft (ServerNetworkProtocolProperty-Klasse)

## <a name="syntax"></a>Syntax  
  
```  
  
object  
.InstanceName [= value]  
```  
  
## <a name="parts"></a>Teile
