# <a name="date-property-clientnetlibinfo-class"></a>Date-Eigenschaft (ClientNetLibInfo-Klasse)
  Ruft das Datum der Clientnetzwerk-Bibliotheksdatei ab.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
object  
.Date [= value]  
```  
  
## <a name="parts"></a>Teile  
 *object*  
 Ein Objekt der [ClientNetLibInfo-Klasse](clientnetlibinfo-class.md) , das Informationen über die Clientnetzwerkbibliothek darstellt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Ein uint32-Wert, der das Datum der Clientnetzwerk-Bibliotheksdatei angibt.
