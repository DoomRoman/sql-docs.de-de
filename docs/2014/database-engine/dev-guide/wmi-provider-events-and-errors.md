# <a name="wmi-provider-events-and-errors"></a>Ereignisse und Fehler für WMI-Anbieter
  Dieses Thema enthält Informationen zu Ursachen und Lösungen für eine Reihe von WMI-Fehlern, die sich auf [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]beziehen.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 Die Themen zu Fehlermeldungen in diesem Abschnitt enthalten eine Erklärung der Fehlermeldung, mögliche Ursachen sowie Maßnahmen, die Sie ergreifen können, um das Problem zu beheben.  
  
 0x8007052f  
 Anmeldefehler: Eingeschränktes Benutzerkonto. Mögliche Ursachen hierfür sind, dass leere Kennwörter nicht zulässig sind, Einschränkungen der Anmeldezeiten vorliegen oder eine Richtlinieneinschränkung erzwungen wurde.
