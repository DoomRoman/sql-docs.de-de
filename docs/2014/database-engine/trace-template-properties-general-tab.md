# <a name="trace-template-properties-general-tab"></a>Eigenschaften der Ablaufverfolgungsvorlage (Registerkarte Allgemein)
  Mithilfe der Registerkarte **Allgemein** des Dialogfelds **Eigenschaften der Ablaufverfolgungsvorlage** können Sie neue Ablaufverfolgungsvorlagen mithilfe der folgenden Optionen erstellen. Um auf dieses Dialogfeld zuzugreifen, zeigen Sie in [!INCLUDE[ssSqlProfiler](../includes/sssqlprofiler-md.md)] **Datei** auf **Vorlagen**, und klicken Sie dann auf **Neue Vorlage**.  
  
## <a name="options"></a>Tastatur  
 **Servertyp auswählen**  
 Geben Sie den Servertyp an, für den die Vorlage verwendet wird.  
  
 **Name der neuen Vorlage**  
 Geben Sie einen aussagekräftigen Namen für die Vorlage an.  
  
 **Neue Vorlage auf vorhandener basieren**  
 Verwenden Sie eine Vorlage aus der Liste als Basis für diese Vorlage. Alle ausgewählten Ereignisse, Datenspalten und Filter stimmen anfangs mit denen in der vorhandenen Vorlage überein und können dann nach Bedarf geändert werden.  
  
 **Als Standardvorlage für den ausgewählten Servertyp verwenden**  
 Verwenden Sie diese Vorlage standardmäßig für alle Ablaufverfolgungen, die für diesen Servertyp erstellt werden.
