# <a name="set-advanced-properties-ssas"></a>Erweiterte Eigenschaften festlegen (SSAS)
  Auf dieser Seite des **Tabellenimport-Assistenten** können Sie zusätzliche Eigenschaftseinstellungen für eine Datenquellenverbindung angeben. Um im [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)]auf den Assistenten zuzugreifen, klicken Sie im Menü **Modell** auf **Aus Datenquelle importieren**.  
  
## <a name="uielement-list"></a>Liste der Benutzeroberflächenelemente  
 **Anbieter**  
 Wählen Sie den Anbieter aus, der für die Datenquellenverbindung verwendet werden soll. Der entsprechende Anbieter muss auf dem Computer installiert sein, um eine Verbindung mit einer Datenquelle herzustellen.  
  
 **Verbindungszeichenfolge**  
 Die Einstellungen, die der Modell-Designer beim Herstellen einer Verbindung mit einer Datenquelle verwendet. Diese Informationen sind bei der Behandlung von Verbindungsproblemen nützlich.  
  
 **Verbindung testen**  
 Stellen Sie unter Verwendung der aktuellen Einstellungen eine Verbindung mit der Datenquelle her. Eine Meldung gibt Aufschluss darüber, ob die Verbindungsherstellung erfolgreich war.
