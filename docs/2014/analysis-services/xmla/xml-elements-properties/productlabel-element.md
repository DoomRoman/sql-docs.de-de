# <a name="productlabel-element"></a>ProductLabel-Element

## <a name="syntax"></a>Syntax  
  
```xml  
  
<Server>  
...  
   <ProductLevel>...</ProductLevel>  
...  
</Server>  
```  
  
## <a name="element-characteristics"></a>Elementmerkmale  
  
|Merkmal|Description|  
|--------------------|-----------------|  
|Datentyp und -länge|Zeichenfolge|
