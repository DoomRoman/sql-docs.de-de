# <a name="go-to-dialog-box-ssas"></a>Gehe zu (Dialogfeld) (SSAS)
  Das Dialogfeld **Gehe zu** ermöglicht das Springen zu einer bestimmten Spalte in einer Tabelle.  
  
## <a name="uielement-list"></a>Liste der Benutzeroberflächenelemente  
  
|||  
|-|-|  
|Begriff|Definition|  
|**Gehe zu**|Wählen Sie eine Spalte aus der Liste **Gehe zu** aus, und klicken Sie anschließend auf **OK**.|
