# <a name="late-breaking-upgrade-issues"></a>Aktuelle Upgradeprobleme
  Die Erkennungslogik wurde möglicherweise erweitert oder geändert, nachdem die Erstellung der Dokumentation abgeschlossen war. Wenn Sie dieses Thema vom Berichts-Viewer des Upgrade Advisors aus aufgerufen haben oder wenn Sie aktuelle Informationen erhalten möchten, lesen Sie die Infodatei, die mit dem Upgrade Advisor installiert wurde.
