# <a name="reporting-services-parameters"></a>Reporting Services-Parameter
  Analysieren [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)], geben Sie eine Instanz von [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] , und geben Sie Informationen zum Herstellen einer Verbindung mit dieser Instanz.  
  
## <a name="options"></a>Tastatur  
 **Reporting Services-Instanzname**  
 Wählen Sie eine Instanz von [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] aus der Liste aus.
