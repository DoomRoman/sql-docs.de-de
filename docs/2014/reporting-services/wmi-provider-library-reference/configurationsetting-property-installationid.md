# <a name="installationid-property-wmi-msreportserverconfigurationsetting"></a>InstallationID-Eigenschaft (WMI: MSReportServer_ConfigurationSetting)
  Diese Eigenschaft gibt einen eindeutigen Bezeichner für eine bestimmte Berichtsserverinstanz zurück.  
  
## <a name="syntax"></a>Syntax  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
