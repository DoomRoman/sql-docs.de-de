# <a name="image-text-box-rectangle-line-f1-help"></a>Hilfe (F1) zu Bild, Textfeld, Rechteck, Zeile
  Die Themen in diesem Abschnitt enthalten Informationen zu den Optionen, die Sie in den Dialogfeldern des Berichts-Designers für Bilder, Textfelder, Rechtecke und Zeile festlegen können.  
  
 Um innerhalb des Dialogfelds **Berichtseigenschaften** auf diese Themen zuzugreifen, klicken Sie auf **Hilfe**.
