# <a name="select-the-report-type"></a>Auswählen des Berichtstyps
  Verwenden Sie diese Seite des Berichts-Assistenten, um entweder einen tabellarischen Bericht oder einen Matrixbericht auszuwählen.  
  
## <a name="options"></a>Tastatur  
 **Tabellarisch**  
 Erstellt einen Bericht, der eine Tabelle enthält. Eine Tabelle ist ein Datenbereich, in dem Daten in Zeilen und Spalten angeordnet sind. Tabellen verfügen über einen statischen Satz von Spalten, und die Anzahl der Zeilen in der Tabelle hängt von den Daten im Dataset ab.  
  
 Auf den nachfolgenden Seiten des Assistenten werden Optionen zum Definieren der Tabelle bereitgestellt.  
  
 **Matrix**  
 Erstellt einen Bericht, der eine Matrix enthält. Eine Matrix ist ein Datenbereich, in dem Daten in Zeilen und Spalten angeordnet sind. Matrizen stellen ähnliche Funktionen wie Kreuztabellen und Pivottabellen bereit. Während eine Tabelle jedoch einen statischen Satz von Spalten aufweist, können Matrixspalten dynamisch sein.  
  
 Auf den nachfolgenden Seiten des Assistenten werden Optionen zum Definieren der Matrix bereitgestellt.
