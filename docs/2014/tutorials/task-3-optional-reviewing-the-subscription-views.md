# <a name="task-3-optional-reviewing-the-subscription-views"></a>Aufgabe 3 (optional): Überprüfung der Abonnementsichten
  In dieser Aufgabe bestätigen Sie, dass die SQL-Ansichten mit SQL Server Management Studio erstellt werden.  
  
1.  Starten Sie **SQL Server Management Studio**. Klicken Sie auf die **starten** Schaltfläche, klicken Sie auf **Programme**, klicken Sie auf **Microsoft SQL Server 2012**, und klicken Sie dann auf **SQL Server Management Studio**.  
  
2.  In der **Verbindung mit Server herstellen** fest **Servertyp** auf **Datenbankmodul**, Typ der **Servernamen** (oder wählen Sie **( Local)**, und wählen Sie entsprechende **Authentifizierung**, und klicken Sie auf **verbinden**.  
  
3.  In der **Objekt-Explorer** Bereich, erweitern Sie **Datenbanken**, erweitern Sie **MDS**, und erweitern Sie dann **Ansichten**.  
  
4.  Vergewissern Sie sich, dass Sie sehen die **vor. Lieferanten** Ansicht in der Liste.
